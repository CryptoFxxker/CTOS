# CTOS 量化交易管理系统介绍

CTOS（Cryptocurrency Trading Operating System）是一个功能强大的量化交易管理系统，专为加密货币交易者设计。

## 系统概述

CTOS 系统提供了完整的量化交易解决方案，包括：

- **多交易所支持**：集成 OKX、Backpack 等主流交易所
- **实时监控**：账户余额、持仓信息实时更新
- **策略管理**：可视化策略配置和监控
- **技术分析**：丰富的技术指标和图表工具
- **资讯中心**：及时的市场动态和深度分析

## 核心特性

### 1. 账户管理

系统支持多个交易所账户的统一管理，可以：

- 实时查看所有账户余额
- 监控账户健康状态
- 查看详细的仓位和订单信息
- 支持账户数据的自动刷新

### 2. 指标分析

提供多种技术指标分析工具：

- **TOPDOGINDEX 指标**：多时间框架对比分析
- **全币种趋势**：市场整体趋势分析
- **K线图表**：支持多种币种和时间间隔

### 3. 策略执行

- 策略参数可视化配置
- 实时监控策略运行状态
- 查看策略运行日志
- 错误信息追踪

## 技术架构

CTOS 系统采用现代化的技术栈：

- **后端**：Django 4.x + Python 3.8+
- **前端**：HTML5/CSS3 + JavaScript ES6+
- **数据库**：SQLite（开发环境）
- **API集成**：OKX API、Backpack API

## 安全特性

- 用户认证和授权系统
- API 密钥安全存储
- 邀请码注册机制
- 管理员权限管理
