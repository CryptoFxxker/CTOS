# CTOS 系统更新日志

本文档记录了 CTOS 量化交易管理系统的主要更新内容。

## 版本 1.3.0 (2024-12-15)

### 🎉 新功能

#### 资讯中心
- ✨ 新增资讯板块，支持博客式文章展示
- ✨ 支持 Markdown 文件自动导入
- ✨ 文章搜索和分类筛选功能
- ✨ 美观的文章展示界面，支持图床背景

#### 账户管理
- ✨ 优化账户详情页面显示
- ✨ 改进数据刷新机制

### 🔧 改进

- 📝 优化用户界面，提升用户体验
- 🐛 修复多个已知问题
- ⚡ 提升系统响应速度
- 📱 改进移动端适配

### 📚 文档

- 新增多篇使用指南和教程文章
- 完善系统功能说明文档
