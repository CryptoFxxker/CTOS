# 账户管理功能使用指南

账户管理是 CTOS 系统的核心功能之一，帮助您统一管理多个交易所账户。

## 访问账户管理

1. 登录系统后，点击导航栏中的 **"账户信息"**
2. 或直接访问 `/accounts/` 路径

## 账户列表页面

账户列表页面显示所有已配置的交易所账户，包括：

### 账户信息展示

- **交易所标识**：显示账户所属的交易所（OKX、Backpack 等）
- **账户ID**：账户的唯一标识符
- **总余额**：账户的总资产价值（USDT）
- **健康状态**：账户连接状态（正常/错误/未知）
- **配置状态**：API 配置是否正确

### 操作功能

- **查看详情**：点击"查看详情"按钮查看账户的详细信息
- **刷新全部**：点击"刷新全部"按钮更新所有账户数据
- **自动刷新**：可以开启自动刷新功能，系统会定期更新数据

## 账户详情页面

点击"查看详情"后，您可以查看：

### 1. 账户概览

- 账户基本信息
- 总资产统计
- 账户健康状态

### 2. 持仓信息

显示当前所有持仓：

- **交易对**：如 BTC/USDT
- **持仓方向**：做多（Long）或做空（Short）
- **持仓数量**：当前持仓数量
- **持仓价值**：持仓的 USDT 价值
- **未实现盈亏**：当前持仓的盈亏情况

### 3. 订单信息

显示最近的订单记录：

- **订单ID**：订单的唯一标识
- **交易对**：订单的交易对
- **订单方向**：买入（Buy）或卖出（Sell）
- **订单类型**：限价单、市价单等
- **订单状态**：已成交、已取消、待成交等
- **成交价格**：订单的成交价格
- **成交数量**：订单的成交数量

## 账户状态说明

### 健康状态

- **正常**：账户连接正常，可以正常获取数据
- **错误**：账户连接失败，可能是 API 配置问题或网络问题
- **未知**：无法确定账户状态

### 配置状态

- **OK**：API 配置正确
- **错误**：API 配置有问题，需要检查

## 常见问题

### Q: 账户显示"不健康"怎么办？

A: 请检查：
1. API 密钥是否正确配置
2. 网络连接是否正常
3. 交易所 API 是否可用

系统会自动重试连接，如果问题持续，请联系管理员。

### Q: 如何添加新账户？

A: 新账户需要在系统配置文件中添加，请联系系统管理员进行配置。

### Q: 数据多久更新一次？

A: 默认情况下，您可以手动刷新。如果开启了自动刷新，系统会每 10 秒更新一次数据。

## 最佳实践

1. **定期检查账户状态**：确保所有账户都处于"正常"状态
2. **关注账户余额变化**：及时发现异常情况
3. **查看持仓详情**：了解当前的风险敞口
4. **监控订单执行**：确保策略正常执行
